=========


## Development

### Preparation
1. Read [package.json](./package.json)
2. Read [.travis.yml](./.travis.yml)
3. Read [env](./env)
4. Read [conf](./conf)

### Installation
```
git clone git@github.com:kjirou/si_of_sis.git
cd si_of_sis
npm install
```

### Run Web Server
- 1. `mongod`
- 2. `npm run dev`

### Use Mongo Script
- `mongo db_name scripts/mongo/script_name.js`
